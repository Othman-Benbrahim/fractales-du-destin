# Fractales du Destin — Oracle Algorithmique IRIS∞

> *Un miroir symbolique. Pas un verdict.*

Skill pour Claude Code — Oracle d'intuition structurée basé sur le Tarot Fractal IRIS∞ (84 cartes, 9 familles).

---

## Installation

### Option 1 — Cloner dans votre dossier skills

```bash
git clone https://github.com/VOTRE_NOM/fractales-du-destin ~/.claude/skills/fractales-du-destin
```

### Option 2 — Copie manuelle

1. Téléchargez ce dépôt (bouton **Code → Download ZIP**)
2. Décompressez et copiez le dossier `fractales-du-destin` dans `~/.claude/skills/`

### Structure attendue

```
~/.claude/skills/
└── fractales-du-destin/
    ├── SKILL.md                    ← Fichier principal (oracle complet)
    └── references/
        ├── arcanes.md              ← Les 84 cartes détaillées
        └── protocoles.md          ← Protocoles de sécurité complets
```

---

## Utilisation

Une fois installé, lancez Claude Code dans n'importe quel projet et demandez :

```
Lance un tirage Fractales du Destin sur [votre situation]
```

Ou simplement :

```
Tirage oracle — je traverse une période de transition professionnelle
```

Le skill se déclenche automatiquement pour toute demande de tirage symbolique, lecture d'arcanes ou consultation oracle.

---

## Ce que fait ce skill

- **Tirage narratif cyclique** : 5 cartes (Nœud de Destin · Bifurcation · Résonance · Ancrage · Miroir)
- **Double voix** : Oracle poétique + Gardien critique
- **9 familles d'arcanes** : Origines 🌬 · Transformation 🔥 · Flux 🌊 · Ancrage 🌍 · Vide Ø · Fractales ♾ · Visionnaires 👁 · Évolution 🔮 · IRIS∞ ✦
- **Protocoles de sécurité** : anti-surinterprétation · Contre-Signe · biais cognitifs
- **Gabarits de réponse** : lecture courte · tirage complet · décision · collectif

## Ce que ce skill ne fait pas

- Prédire l'avenir
- Diagnostiquer une situation médicale, juridique ou financière
- Lire l'intériorité d'une personne absente
- Produire des certitudes

## Fondements

Système conçu autour du **Codex IRIS∞** et de ses 9 modules algorithmiques :
CMN · PNB · CNI · DSF · TSF · REI · IDI · PRA · RCL

Et du module introspectif **TRANSCOM IRIS1414** — écoute des signaux faibles internes.
