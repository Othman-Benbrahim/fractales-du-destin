# Module du Contre-Signe

## Fonction

Le Contre-Signe introduit une résistance dans l’interprétation.
Il protège le joueur contre les fausses évidences symboliques.

## Quand l’activer ?

Le module s’active lorsque :

- une interprétation paraît trop parfaite ;
- le joueur cherche une confirmation à tout prix ;
- plusieurs signes semblent converger trop rapidement ;
- l’émotion est très forte ;
- aucune preuve réelle n’est disponible ;
- l’Oracle détecte un risque de surinterprétation.

## Cartes du Contre-Signe

### 1. Le Faux Écho

Ce qui ressemble à une synchronicité peut être une répétition mentale.

### 2. Le Miroir Trop Parfait

Ce qui confirme exactement ton attente doit être interrogé.

### 3. La Preuve Absente

Ce qui n’a pas encore de trace ne doit pas être affirmé.

### 4. Le Silence Non Signifiant

Tout silence n’est pas un message.

### 5. La Porte Fermée

Certaines choses doivent rester non interprétées pour l’instant.

### 6. Le Désir Déguisé

Le signe peut être le masque d’un désir.

### 7. La Peur Prophétique

La peur peut fabriquer une fausse prophétie.

## Sortie obligatoire

Quand le Contre-Signe apparaît, l’Oracle doit produire :

1. Ce que le signe semble dire.
2. Pourquoi il peut être trompeur.
3. Ce qu’il faut vérifier.
4. Ce qu’il vaut mieux laisser ouvert.
