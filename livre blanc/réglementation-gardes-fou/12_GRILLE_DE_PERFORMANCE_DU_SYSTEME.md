# Grille de performance du système

## Objectif

Évaluer l’efficacité réelle de Fractales du Destin comme outil liminal d’intuition structurée.

## Critères de performance

Chaque tirage doit être évalué selon 7 critères, de 0 à 5.

### 1. Clarté intérieure

0 : le tirage augmente la confusion  
5 : le tirage clarifie fortement la situation

### 2. Pertinence symbolique

0 : les images semblent arbitraires  
5 : les images résonnent fortement avec le contexte

### 3. Ancrage réel

0 : aucune donnée concrète  
5 : plusieurs indices vérifiables

### 4. Prudence interprétative

0 : interprétation excessive  
5 : distinction claire entre symbole, hypothèse et fait

### 5. Action concrète

0 : aucune action possible  
5 : une action simple et utile apparaît

### 6. Effet émotionnel

0 : augmentation de l’angoisse  
5 : apaisement, lucidité ou recentrage

### 7. Calibration rétrospective

0 : impossible à réévaluer  
5 : possibilité claire de vérifier plus tard

## Score total

/35

## Interprétation

0-10 : tirage faible ou confus  
11-20 : tirage partiellement utile  
21-28 : tirage pertinent  
29-35 : tirage très performant

## Règle

Un tirage performant n’est pas forcément un tirage spectaculaire.
Un bon tirage est un tirage qui clarifie sans enfermer.
