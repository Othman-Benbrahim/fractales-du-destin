
# Prompts opérationnels pour l’Oracle Algorithmique

## Prompt 1 — Lecture liminale simple

Analyse la situation selon le protocole liminal :
1. Visible
2. Voilé
3. Seuil
4. Contre-Signe
5. Ancrage

Pour chaque niveau, distingue :
- image poétique ;
- hypothèse intuitive ;
- indice vérifiable ;
- risque de projection ;
- action minimale.

## Prompt 2 — Analyse d’un signe

Analyse ce signe sans le surinterpréter.
Classe-le selon :
- image poétique ;
- résonance subjective ;
- hypothèse intuitive ;
- indice réel ;
- niveau de prudence.

Ajoute une lecture alternative.

## Prompt 3 — Calibration d’un tirage passé

Reprends ce tirage et compare-le aux événements observés.
Classe le résultat :
- confirmé ;
- partiellement confirmé ;
- infirmé ;
- non vérifiable ;
- utile symboliquement mais non factuel.

Déduis une leçon de calibration.

## Prompt 4 — Activation du Gardien critique

Relis l’interprétation précédente.
Repère :
- les biais possibles ;
- les affirmations trop fortes ;
- les confusions entre symbole et preuve ;
- les hypothèses alternatives.

Réécris ensuite une version plus juste.

## Prompt 5 — Synthèse d’action

À partir du tirage, propose :
1. une phrase symbolique ;
2. une hypothèse concrète ;
3. une action minimale ;
4. une chose à ne pas faire ;
5. un point à observer pendant 7 jours.
